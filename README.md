# じかんわり — 学校時間割アプリ

HTML / CSS / JavaScript だけで作った、PWA対応の学校時間割アプリです。

## 使い方
1. `index.html` を開く（推奨: 下記の方法でホスティングしてから開く）
2. 右上「専攻を選ぶ」から教科プリセット（普通科／理系専攻／文系専攻／商業専攻／芸術専攻）を選択
3. 左側の教科カードを時間割のマスにドラッグ&ドロップして登録
4. マスをクリックすると、教室・持ち物・カードの色を編集できます
5. 右上「通知オフ」ボタンで通知を許可すると、授業開始10分前に持ち物をお知らせします
6. 「週表示／月表示」で切り替え可能。月表示で日付をクリックするとその日の全時間割が見られます

## 時間割の仕様
- 月・水・金：6時限　／　火・木：7時限
- 8:50開始、1コマ50分、休み時間10分
- 4時限と5時限の間は40分の昼休み

## PWA化（ホーム画面に追加）
このフォルダ一式（index.html, style.css, app.js, manifest.json, sw.js, icons/）を
HTTPSでホスティングしてください（GitHub Pages, Netlify, Vercel など。ローカルの `file://` では
Service Workerが動かないため、`npx serve` 等のローカルサーバーでもOKです）。

ブラウザでアクセス後、スマホなら「ホーム画面に追加」、PCならアドレスバーのインストールアイコンから
アプリとしてインストールできます。

## 通知についての注意
ブラウザの通知APIは、**アプリ（タブ）を開いている間**に10分前チェックを行い通知を出す仕組みです。
アプリを閉じた状態でも通知を届けるには、本来サーバー側からのプッシュ通知（Web Push）と
バックエンドの時刻管理サーバーが必要になりますが、今回は静的サイトのみの構成のため、
「アプリを開いている間は自動で持ち物をお知らせする」仕様にしています。
（PWAとしてインストールし、バックグラウンドタブとして開いたままにしておくと通知が届きます）
